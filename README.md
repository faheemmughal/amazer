# amazer
a maze generator and solver using recursive backtracker algorithm


##### Installation & Execution

* Get all dependecies: `bundle`
* Run tests: `rspec`
* Run: `./amaze_me.rb`



##### Project Layout

* bin/*  : Executables
* lib/*.rb : Source files
* spec/*_spec.rb : test files
