# amazer
a maze generator and solver


== Installation & Execution

* Get all dependecies: `bundle`
* Run tests: `rspec`
* Run: `ruby lib/amazer.rb`

== Project Layout

* lib/*.rb : Source files
* spec/*_spec.rb : test files
